### Architecture
**Fichier principal**: `src/climax_core/simple_adaptive_mask.py`

**Design**:


**Fixed Physics Component** (non-learnable):

### Comment √ßa marche

**√âtape 1**: Input contient elevation
```python

```

**√âtape 2**: Downsample to patches
```python

```

**√âtape 3**: Compute pairwise differences
```python

elev_j = elev_patches.unsqueeze(1)  # [B, 1, 8192]

# elev_diff[b,i,j] = elevation patch j - elevation patch i
```

**√âtape 4**: Apply physics
```python

# Strong wind reduces barrier

# Scale by learnable alpha

```

**√âtape 5**: Apply in attention
```python

# Negative bias ‚Üí reduces attention probability i‚Üíj
```
