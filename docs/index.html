TopoFlow

A Vision Transformer model for multi-pollutant air quality forecasting with wind-following attention and terrain-aware modeling.

View on GitHub Get Started

Key Innovations

TopoFlow introduces physics-informed inductive biases into the Vision Transformer architecture for atmospheric modeling.

🌬️

Wind-Following Attention

Patches are reordered based on wind direction, creating an inductive bias aligned with atmospheric transport (upwind → downwind).

⛰️

Elevation-Aware Modeling

Learnable topographic barrier modeling penalizes attention across elevation gradients, capturing mountain blocking effects.

🎯

Multi-Horizon Prediction

Simultaneous forecasting at 12h, 24h, 48h, and 96h lead times with a single forward pass.

🧪

Multi-Pollutant Support

Joint prediction of PM2.5, PM10, SO₂, NO₂, CO, and O₃ with shared representations.

⚡

Efficient Architecture

Bucketed relative position bias for O(N) memory instead of O(N²), enabling large-scale training.

🔧

Easy Integration

Built on PyTorch Lightning with clean APIs. Install with pip and start training in minutes.

Architecture

TopoFlow extends the Vision Transformer with physics-informed components in the first attention block.

Input
[B, 15, 128, 256] — 15 variables (meteo + pollutants + terrain)
↓
Wind-Guided Patch Embedding
Variable tokenization + wind-following reordering
↓
TopoFlow Block (Block 0)
Relative 2D position bias + α elevation mask
↓
Standard ViT Blocks (Blocks 1-5)
Multi-head self-attention + MLP
↓
Decoder Head
[B, 6, 128, 256] — 6 pollutants output

Quick Start

Get started with TopoFlow in just a few lines of code.

# Installation pip install git+https://github.com/AmmarKheder/TopoFlow.git # Training from topoflow import TopoFlowModel, TopoFlowDataModule model = TopoFlowModel( variables=["u", "v", "temp", "pm25", ...], img_size=(128, 256), embed_dim=768, depth=6, ) # Run inference predictions = model(x, lead_times, variables)
